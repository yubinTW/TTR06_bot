# TTR06_bot
Telegram bot made by team #6 of 2019 SITCON Camp

## Before you run this bot

### Setup env

* Install Python 3
* Install `pipenv`
```
$ pip install pipenv
```
* Install deps
```
$ pipenv install
```

### Run the bot

```
$ pipenv run python3 ./app/main.py
```

## feature:
- 文字特效
- 你天殺的忘記切輸入法
